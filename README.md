# Shell Scripting Practice

This repository contains my hands-on practice while learning Bash shell scripting for DevOps and Linux automation.

## Topics Covered

### Bash Basics
- Shebang (`#!/bin/bash`)
- Variables
- User input using `read`
- Command line arguments (`$1`, `$2`)
- Environment variables
- Output using `echo`

### Conditional Statements
- `if`, `elif`, `else`
- Numeric comparisons
- String comparisons
- Nested conditions
- Input validation using regex

### Loops
- `for` loops
- `while` loops
- Arithmetic operations
- Odd/even number logic

### Functions
- Creating functions
- Passing arguments to functions
- Reusable scripting logic

### File & Directory Operations
- File existence checks
- Directory creation
- Automatic file creation
- Exit status handling using `$?`
- Logical operators (`&&`, `||`)

### Text Processing
- Pipes (`|`)
- `grep`
- `cut`
- `awk`
- `sed`

## Goal

The goal of this repository is to build strong Linux and shell scripting fundamentals required for:
- DevOps
- Automation
- Server management
- Log analysis
- Deployment scripting
